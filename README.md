# Offline Music PWA

**Offline Music PWA** — минималистичный музыкальный плеер на чистых HTML, CSS и JavaScript. Приложение сохраняет выбранные MP3-файлы в IndexedDB и остаётся доступным без интернета после первого открытия благодаря Service Worker.

## Возможности

- интерфейс на русском языке в тёмной теме;
- добавление нескольких MP3-файлов с устройства;
- сохранение треков в IndexedDB между сессиями;
- офлайн-доступ к интерфейсу и сохранённой библиотеке;
- отображение названия файла, длительности и размера;
- удаление отдельного трека и очистка всей библиотеки;
- встроенный аудиоплеер с воспроизведением, паузой, перемоткой, громкостью и прогрессом;
- PWA-манифест и Service Worker без сторонних библиотек или CDN.

## Локальный запуск

Service Worker работает только на `localhost` или по HTTPS. Для проверки локально запустите любой статический сервер из корня проекта:

```bash
python3 -m http.server 8080
```

Затем откройте:

```text
http://localhost:8080/
```

После первого посещения добавьте MP3-файлы, перезагрузите страницу и отключите интернет: интерфейс и сохранённые треки останутся доступными.

## Публикация на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него файлы проекта или привяжите локальный репозиторий:

   ```bash
   git remote add origin https://github.com/<username>/<repository>.git
   git branch -M main
   git push -u origin main
   ```

3. Откройте настройки репозитория: **Settings → Pages**.
4. В блоке **Build and deployment** выберите:
   - **Source**: `Deploy from a branch`;
   - **Branch**: `main`;
   - папка: `/ (root)`.
5. Нажмите **Save**.
6. Через несколько минут приложение будет доступно по адресу:

   ```text
   https://<username>.github.io/<repository>/
   ```

## Важные заметки

- MP3-файлы не отправляются на сервер и хранятся только локально в браузере пользователя.
- При очистке данных сайта в браузере IndexedDB-библиотека будет удалена.
- Для установки PWA используйте меню браузера «Установить приложение» после публикации на HTTPS или запуска на `localhost`.
